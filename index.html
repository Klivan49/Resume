Обо мне

Мой путь в программировании начался примерно 3 года назад. Сначала я изучил основы языка Python, что дало мне представление о скриптовом программировании. Вскоре я освоил C# и занялся созданием игр на Unity вместе с другом, используя GitHub для совместного программирования. Также мы разработалм игровой движок на C++ с использованием библиотеки SFML 3.0.

После поступления в БГУИР на специальность цифровой маркетинг я начал углубленно изучать языки программирования C и C++ при выполнении множества лабораторных работ, что существенно расширило мои знания базовых алгоритмов и структур данных. Этот опыт заложил прочную основу в низкоуровневом программировании. Лучшие проекты доступны в моём профиле на GitHub.

Помимо учебы я помогал преподавателю проверять лабораторные работы по программированию, что позволило мне получить ценный опыт в анализе кода и внимании к деталям. Для ускорения работы над курсовыми проектами я написал на Python программу, которая проверяет и автоматически исправляет оформление курсовой и прилагая файл с описанием изменений. Этот проект также опубликован на моём GitHub.
